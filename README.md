
# DictSQLite

You can handle basic SQLite operations in Python like Dict.

- [日本語のREADMEはこちら](./README_JP.md)

## Installation

Install [DictSQLite](https://pypi.org/project/DictSQLite/) with pip

```bash
  pip install dictsqlite
```
